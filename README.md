## Requirements

* CakePHP 2.4.2 or greater.
* PHP 5.3.10 or greater.

## Installation

Clone/Copy the files in this directory into `app/Plugin/UniLogin`

## Configuration

Ensure the plugin is loaded in `app/Config/bootstrap.php` by calling:
